# 168VSC System User Guide

คู่มือการใช้งานระบบบริหารจัดการ 168VSC ครอบคลุมตั้งแต่นำเข้าสินค้า จนถึงการจัดส่ง

## 1. การจัดการสินค้าและการจัดซื้อ (Supply Chain)

กระบวนการนำเข้าสินค้าเข้าสู่ระบบอย่างถูกต้อง เพื่อให้สามารถตรวจสอบต้นทุนและสต็อกได้แม่นยำ

### 1.1 สร้างสินค้า (New Product)

- ไปที่เมนู **Products**
- คลิกปุ่ม `+ เพิ่มสินค้าใหม่`
- กรอกข้อมูลที่จำเป็น:
  - **รหัสสินค้า (Code)**: *สำคัญมาก* ใช้สำหรับอ้างอิงทั้งระบบ
  - **ราคาขาย (Price)**: ราคาตั้งต้นสำหรับฝ่ายขาย
  - **ตัวเลือก (Variants)**: หากสินค้ามีหลายสี/ไซส์ ให้เพิ่มที่นี่
- *Tip: ไม่จำเป็นต้องใส่จำนวน Stock ในหน้านี้ หากจะมีการรับเข้าผ่าน PO*

### 1.2 สั่งซื้อสินค้า (Purchase Order - PO)

- ไปที่เมนู **Purchasing**
- คลิก `+ New PO`
- **เลือก Supplier**: ร้านค้าหรือโรงงานที่สั่งซื้อ
- **เพิ่มรายการ**: เลือกสินค้าที่สร้างไว้ และระบุจำนวน (Qty)
- **สถานะการเงิน (Financial)**: สามารถระบุได้ว่าจ่ายเงินแล้วหรือยัง (Paid/Unpaid)

### 1.3 การรับสินค้า (Receiving)

เมื่อสินค้ามาถึงโกดัง:

1. หา PO ที่ต้องการในหน้า Purchasing
2. คลิกปุ่ม **Receive (ไอคอนกล่อง)** หรือเข้าไปเปลี่ยนสถานะเป็น `Received`
3. ระบบจะสร้าง **Inventory Items** ให้อัตโนมัติ โดยอ้างอิง Lot Number ตามเลขที่ PO
4. สต็อกสินค้าจะเพิ่มขึ้นในระบบทันที

---

*รออัปเดตส่วนถัดไป: การขาย (Sales) และการจัดส่ง (Shipping)*
